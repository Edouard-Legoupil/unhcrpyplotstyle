v0.2.0 (25/06/2026)
====================
- **Major Update**: Complete revision to comply with UNHCR Data Visualization Guidelines (June 2025)
- **Typography Updates**:
  - Title size: 14pt → 12pt bold
  - Label size: 10pt → 8pt
  - Legend font size: 10pt → 8pt
  - Color updates: Title (#000000), Labels (#4D4D4D), Legend (#4D4D4D)
- **Color Palette Updates**:
  - Base color cycle: Updated to official UNHCR sequential blue palette
  - Sequential colors: All updated to official 6-step ramps (blue, green, yellow, red, cyan, purple)
  - Diverging colors: Updated blue-red palette with official UNHCR colors
  - Category colors: Updated region and PoC palettes with official UNHCR colors
- **New Color Files**:
  - Added `sequential_cyan.mplstyle` with official UNHCR cyan ramp
  - Added `sequential_purple.mplstyle` with official UNHCR purple ramp
- **Grid & Elements**:
  - Grid color: #999999 → #BFBFBF (25% black)
  - Legend color: #333333 → #4D4D4D (85% black)
- **Compliance**: All colors now meet UNHCR AAA accessibility standards
- **Backward Compatibility**: Maintained - existing code continues to work with updated visuals
